Папка для ваших фотографий работ.
Вы можете загрузить сюда файлы картинок (например: dawn_reflection.jpg, silent_forest.jpg, urban_geometry.jpg) через GitHub или файловый менеджер.
Сайт автоматически найдет их через Vite glob-import и покажет в выпадающем списке в админ-панели при добавлении новой работы!
